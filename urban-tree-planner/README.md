# Urban Tree Planner

## Overview

**Urban Tree Planner** is a multi‑agent application built with the Google Agent Development Kit (ADK). It demonstrates a simple orchestration pattern where a top‑level orchestrator agent decides, based on the user query, whether to fetch simulated weather information or the current time for a given location. The system includes:

- `weather_agent` – provides mock weather data.
- `time_agent` – returns the current time for a city.
- `orchestrator` – sanitises input, determines the appropriate sub‑agent, and delegates the request.
- Security checkpoint – basic input sanitisation and prompt‑injection detection.

The project is scaffolded using ADK’s `agents-cli` and can be run locally via `adk web`.

## Architecture

```
+-------------------+      +-------------------+      +-------------------+
|   Orchestrator    | ---> |  Weather Agent    |      |   Time Agent      |
| (LlmAgent)        |      | (LlmAgent)        |      | (LlmAgent)        |
+-------------------+      +-------------------+      +-------------------+
        |  ^                     |  ^
        |  |                     |  |
        |  +--- AgentTool ------+  +--- AgentTool -----+
        |
        +--- Security Checkpoint (function)
```

The orchestrator uses **`AgentTool`** to wrap the sub‑agents, allowing the LLM to call them as functions.

- Python 3.12 (or compatible)

## Setup

```bash
# Clone the repository (if not already done)
git clone <repo-url>
cd urban-tree-planner

# Install dependencies (the scaffold already created a .venv)
uv sync

# Activate the virtual environment (optional – uv does this automatically)
uv run bash
```

## Running the Application

```bash
# Start the ADK dev server (UI available at http://127.0.0.1:18081)
uv run adk web --port 18081
```

Open the **Dev UI** in your browser, select the `app` application, and type a prompt such as:

- `What is the weather in Bangalore?`
- `What time is it in Tokyo?`

The orchestrator will sanitize the input, decide which sub‑agent to invoke, and return a response.
